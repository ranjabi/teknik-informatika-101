### Tugas Besar - Raft Protocol
Implementasi raft protocol.
#### Additional Info for the Developer
- Komunikasi menggunakan XMLRPC pada python
- Library yang digunakan adalah xmlrpc, json, asyncio, time, socket, threading, dan sys
- Address (IP dan port) akan menjadi identitas setiap node
- Implementasi awal adalah server-client dengan RPC sederhana
- Mengimplementasikan membership change
- Mengimplementasikan log replication
- Mengimplementasikan heartbeat
- Mengimplementasikan leader election
#### Useful links and resources
- [Raft Paper - Github Pages](https://raft.github.io/)
- [Original Raft Paper](https://raft.github.io/raft.pdf)
- [Raft visualization](http://thesecretlivesofdata.com/raft/)
- [Raft Wikipedia](https://www.wikiwand.com/en/Raft_(algorithm))
- [distribution system notes](https://www.cl.cam.ac.uk/teaching/2122/ConcDisSys/dist-sys-notes.pdf)

##	IF3260 Grafika Komputer
### Tugas Besar 1 - WebGL 2D Web Based Computer Aided Design (CAD)  
Website dengan fitur menggambar dan mengedit model pada kanvas menggunakan WebGL murni, tanpa library/framework tambahan. Model terdiri dari garis, persegi, persegi panjang, dan polygon.
#### User stories
  - [ ] Pengguna dapat menggambar model garis, persegi, persegi panjang, dan polygon
  - [ ] Pengguna dapat mengubah panjang garis
  - [ ] Pengguna dapat mengubah panjang sisi persegi
  - [ ] Pengguna dapat mengubah panjang atau lebar persegi panjang
  - [ ] Pengguna dapat menambah dan menghapus titik sudut
  - [ ] Pengguna dapat melakukan transformasi geometri (pilih 2 dari translasi, dilatasi, rotasi, shear)
  - [ ] Pengguna dapat menggerakkan titik sudut dengan drag and drop
  - [ ] Pengguna dapat mengubah warna titik sudut
  - [ ] Pengguna dapat menyimpan dapat menyimpan dan memuat model yang dibuat
#### Useful links and resources
- [WebGL Fundamentals](https://webglfundamentals.org/)
#### Example projects
- [Repository](https://github.com/ranjabi/2d-web-based-cad)  

### Tugas Besar 2 - WebGL 3D Hollow Object  
Website visualisasi 3D hollow object merupakan versi tiga dimensi dari tugas besar 1. Kali ini model yang dibuat adalah objek berongga (hollow objects). 
#### User stories
  - [ ] Pengguna dapat mengubah jenis proyeksi objek (orthographic, oblique, atau perspective)
  - [ ] Pengguna dapat melakukan rotasi, translasi, dan scaling pada objek
  - [ ] Pengguna dapat menggubah jarak view kamera untuk menjauhi, mendekati, dan mengitari objek
  - [ ] Pengguna dapat menyimpan dan memuat model
  - [ ] Pengguna dapat mengaktifkan dan menonaktifkan shading pada model
  - [ ] Pengguna dapat menjalankan animasi pada model
#### Useful links and resources
- [WebGL Fundamentals](https://webglfundamentals.org/)
#### Example projects
- [Repository](https://github.com/ranjabi/IF3260_Tugas2_K01_G18)
